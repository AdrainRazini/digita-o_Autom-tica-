Tutorial Avançado

Como o site foi feito?

Este gerador de scripts VBS foi desenvolvido inteiramente com tecnologias web modernas e funciona 100% no navegador, sem a necessidade de servidor backend.

  • HTML para estruturação do conteúdo.
  • CSS com variáveis :root para estilização e responsividade.
  • JavaScript modular para gerar dinamicamente o arquivo VBS e simular digitação humana.
  • AOS (Animate On Scroll) para animações suaves enquanto o usuário navega.
  • Font Awesome para adicionar ícones que tornam o conteúdo mais visual.

O foco é oferecer uma ferramenta prática, leve e acessível para todos que desejam automatizar digitação no Windows.

Como funciona na prática?

O site permite que o usuário insira um texto personalizado, defina a velocidade de digitação, e gere um arquivo .vbs que automatiza a escrita em qualquer campo do Windows. A geração funciona da seguinte forma:

  1. O texto é higienizado e convertido com cleanText() para evitar caracteres que quebrem o script.
  2. Um código VBS é montado em tempo real e armazenado em um Blob.
  3. Esse Blob é transformado em um link de download, e o navegador automaticamente faz o download do arquivo pronto.
  4. Ao executar o script, ele espera 3 segundos e então começa a digitar caractere por caractere.

Tudo é feito no próprio navegador do usuário, garantindo segurança, privacidade e independência de conexão com servidores.

Curiosidades técnicas

  • Tipagem automática: a simulação de digitação é feita com delays aleatórios para parecer mais humana.
  • Execução instantânea: o arquivo gerado pode ser usado imediatamente após o download.
  • Compatibilidade: o script funciona em qualquer sistema Windows com suporte a VBS (praticamente todos).
  • Leveza: todo o site tem menos de 100KB e carrega em poucos segundos.
